# 石大生存指南

最近阅读了《银河系搭车客指南》，突然再次燃起了写一份石大探店指南的欲望（虽然两者没有什么关系）。

一些同学几乎没怎么出过校门，这多少有点遗憾，其实就三五分钟的步程，就能大大改善大学生活质量。

比如我突然发现荟萃食堂二楼的猪脚饭挺好吃一样（谨慎添加小菜，很辣），吃猪脚饭没有难度，这也不是什么难获取的信息，但有时候就是不知道。

可惜四年时间依然没能吃遍食堂，这个指南的用处就是给大家省点事，不用一个个窗口吃过去，提前点亮新地图，降低探索难度。

学校门口的饭店跟校内食堂消费竟然差不多......

由于前期很多区域不敢探索，走过的地方还是比较少，很多地点都没解锁，如果同学们有去过的，可以补充。

---

突然发现能写的还挺多的......

石大探店指南扩张为石大生存指南......
